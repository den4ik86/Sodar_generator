# Sodar generator
Данный проект создан по ТЗ от НИИ оптики атмосферы г.Томск.  

Руководитель проекта: Камардин Андрей Павлович.  
Руководитель лаборатории группы акустики атмосферы: Оденцов Сергей Леонидович.  

В данном проекте использван принцип амплитудной модуляции сигнала по огибающей окна тьюки.
Проект построен на микроконтроллере LGT8F328P. Контроллер управляет генераторами AD9833 задавая несущую частоту акустического сигнала зондирования в 3 каналах, а ЦАП контроллера формирует фронты нарастания и затухания имульса согласно математическим правилам окна тьюки. Для формирования огибающей используются прерывания по таймеру, для того чтобы строго контролировать процесс формирования огибающей.
Так же контроллер формирует TTL сигналы для коммутации режимов приём/передача, согласно техническому заданию на разработку прибора.  
Для ускорения процесса разработки и макетирования прибора использована среда разработки Arduin IDE, а так же библиотеки:  
AD9833.h - для управления генераторами синусоиды.  
<LiquidCrystal_I2C.h> - для визуализации режима работы и настройки прибора.  
"math.h" - используестся для математических операций используемых в коде (sin/cos)  
Проект содержит схемы двух вариантов амплитудных модуляторов низкой частоты, постороеных на разной элементарной базе, в состав которых обязательно входит ФНЧ на выходе, для компенсации искажений привносимых в сигнал работой 8-ми битного ЦАП микроконтроллера и других паразитных составляющих сигнала.
