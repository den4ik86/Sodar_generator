# Sodar generator
Данный проект создан по ТЗ от НИИ оптики атмосферы г.Томск.
Руководитель проекта: Камардин Андрей Павлович.
Руководитель лаборатории группы акустики атмосферы: Оденцов Сергей Леонидович.
Вданном проекте использван принцип амплитудной модуляции сигнала по огибающей окна тьюки.
Проект построен на микроконтроллере LGT8F328P. Контроллер управляет генераторами AD9833 задавая несущую частоту акустического сигнала зондирования в 3 каналах, а ЦАП контроллера формирует фронты нарастания и затухания имулься согласно математическим правилам окна тьюки. Для формирования огибающей используются прерывания по таймеру, для того чтобы строго контролировать процесс формирования огибающей.
Так же контроллер формирует TTL сигналы для коммутации режимов приём/передача, согласно техническому заданию на разработку прибора.
